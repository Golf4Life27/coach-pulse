# AKB Real-Estate-Tech Gap Analysis v1

**Document version:** v1.0
**Authored:** 2026-05-20 (Code — outside-system architectural review)
**Source spec:** `agents/specialized-domains/real-estate-tech.md` in `rohitg00/awesome-claude-code-toolkit`
**Branch:** `claude/fix-token-burn-cost-JUDad`
**Scope:** Honest gap inventory. No fixes proposed. No solutions designed. Capability-by-capability mapping of an outside spec's framing vs. what Inevitable currently ships.

The source spec is written for a **consumer-facing MLS property platform** (RETS/RESO feeds, faceted search, map display, saved-search notifications to home buyers, hedonic regression AVM). Inevitable is a **wholesale dispatch system** (agent contact, dispo-to-investor-buyers, contract assignment). Where the two frames align, the gap is real; where they diverge, the spec capability is logged as "Doesn't apply" with reasoning.

## Lost-Phone Test definition

For column 4, a capability "maps to Lost-Phone Test" when it must be present-and-autonomous for a deal to move from intake → outreach → reply → negotiation → contract → close **without Alex touching his phone**. Capabilities that improve operator-facing UX, consumer-side experience, or accounting hygiene are marked N — they don't gate the Lost-Phone Test.

## Where the spec genuinely surfaces a gap Inevitable should consider

(Honest read — these are the Lost-Phone-relevant items where the spec's framing exposes something Inevitable hasn't built. Not recommendations; just where the gap matrix points.)

1. **USPS-spec address normalization** — only matters when Crawler 2.0 multi-source dedup lands. Phase 13.6 / Phase 21.6 timing.
2. **Event-driven notification (sub-5-min)** — Phase 21.13 Cadence_Queue substrate decision is the architectural fork. Until then, briefing-pull pattern is the floor.
3. **Hedonic regression on comp adjustment** — quality multiplier on Phase 4A.1 AVM. Current single-multiplier model is sufficient for V2.1 / Phase 4D; regression upgrade is a future tier.
4. **Cross-source dedup fixtures** — quality lock for when Crawler 2.0 lands; not gating until then.
